# ClipMaker-IA

## Project Description
ClipMaker-IA is a project designed to make the process of clipping and editing videos simpler and more efficient. With a user-friendly interface and advanced functionalities, ClipMaker-IA aims to cater to both novice and professional users in the video editing community.

## Pages
- **Home Page**: The landing page that gives an overview of the project.
- **Editor Page**: Where users can upload their videos and access editing tools.
- **Resources Page**: Contains tutorials and help guides for users.

## Functions
- Users can upload video clips easily.
- Provides various editing tools such as cut, merge, and filter effects.
- Options to export videos in different formats.

## Technologies Used
- **Frontend**: React, HTML, CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB
